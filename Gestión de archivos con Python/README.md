#Gestión de archivos con Python. 

- Proyecto práctico para demsotrar el uso de **Python** en la lectura, limpieza y gestión de los datos procedentes de distintas fuentes. 

- El objetivo es mostrar cómo trabajar con archivos **TXT, CSV, Excel** y datos obtenidos mediante **web scraping**, integrándolos en un flujo de trabajo sencillo y reproducible.

-----------------------------
##Objetivos del proyecto. 

- Leer los distintos tipos de archivos usando Python. 
- Convertir la información en un **DataFrame** de Pandas. 
- Realizar una inspección y limpieza básica de los datos. 
- Mostrar una estructura del proyecto clara y ordenada. 
- Aplicar buenas prácticas de trabajo con Git y Github. 

-----------------------------

##Visualización de datos

El proyecto incluye visualizaciones básicas realizadas con `matplotlib` para:
- comparar valores agregados
- analizar distribuciones
- apoyar la interpretación de los datos
